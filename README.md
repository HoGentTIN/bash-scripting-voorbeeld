# Een Bash-script schrijven

Deze repo bevat een uitgewerkt voorbeeld van hoe je een Bash-script schrijft.

In deze README vind je de opgave en telkens de volgende stap die je moet zetten. De bedoeling is dat je het script zelf schrijft in een andere directory. Je kan hier telkens naar de oplossing van de volgende stap komen kijken. Instructies daarvoor vind je verderop.

## Aan de slag

Voer in een Bash terminal de stappen hieronder uit. Je kan dit in eerder welke directory uitvoeren, maar we zullen in de instructies veronderstellen dat je in een directory `~/linux/` zit.

1. Download deze opgave:

    ```console
    git clone https://github.com/hogenttin/bash-scripting-voorbeeld opgave
    ```

2. Maak een directory voor jouw oplossing:

    ```console
    mkdir oplossing
    ```

    Je kan hier zelf ook een Git repository van maken (met `git init oplossing/`) zodat je je eigen tussenstappen kan opvolgen.

3. Ga naar de opgave.

    ```console
    cd opgave
    ```

4. De verschillende tussenstappen zijn in de Git-historiek aangeduid met een "tag": `start`, `stap-1`, `stap-2`, enz. Je vindt de uitleg voor elke stap telkens in README.md. Merk op dat de inhoud bij elke stap verandert! Op het einde staat telkens de instructie om de oplossing voor de volgende stap te bekijken.

    Om te beginnen doe je eerst het volgende:

    ```console
    $ git checkout start
    ```

Veel succes!
